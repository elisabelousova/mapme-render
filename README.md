# Saved Places Bot

MVP Telegram-бота, который принимает ссылки, текст или список мест и собирает личную карту путешественника по городам.

## Что уже работает

- Telegram-бот через long polling;
- прием обычного текста, ссылок и многострочных списков;
- попытка распознать Instagram-ссылки через публичные метаданные;
- AI-извлечение мест из описания и картинки при наличии `OPENAI_API_KEY`;
- эвристическое выделение города, названия места и категории;
- сохранение мест в локальный JSON;
- личная веб-страница пользователя по приватной ссылке;
- группировка мест по городам;
- Google Maps-ссылка для каждого места;
- ручное добавление мест через веб-страницу.

## Что сознательно упрощено в MVP

- бот не логинится в Instagram и не обходит приватность закрытых постов;
- если Instagram не отдаёт подпись или превью, бот попросит прислать подпись, скрин или город;
- звук Reels не распознаётся, пока у нас нет доступного аудио/видеофайла;
- база данных пока JSON-файл, чтобы можно было быстро проверить идею без сервера.

## Запуск

1. Создать бота через `@BotFather`.
2. Скопировать `.env.example` в `.env`.
3. Вставить `BOT_TOKEN`.
4. Опционально вставить `OPENAI_API_KEY`, если нужно распознавать места через AI и OCR по скринам.
5. Запустить:

```bash
npm start
```

Для запуска только веб-интерфейса без Telegram-бота:

```bash
npm run web
```

Локальный адрес: `http://127.0.0.1:8787`.

## Команды

- `/start` — старт и инструкция;
- `/map` — личная карта;
- `/cities` — города;
- `/places` — последние места;
- `/help` — примеры;
- `/delete YES` — удалить свои данные.
